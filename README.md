# flutter-runner

Run multiple Integration Test with a single command.

```shell
Usage: flutter-runner [OPTIONS]

Options:
  -p, --platform <PLATFORM>  
  -c, --config <CONFIG>      
      --flavor <FLAVOR>      
  -d, --device <DEVICE>      
  -f, --file <FILE>          
  -r, --report <REPORT>      
  -t, --type <TYPE>          [default: integration_test]
  -h, --help                 Print help
  -V, --version              Print version
```
